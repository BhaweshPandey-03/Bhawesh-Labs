# Bhawesh Labs

Bhawesh Labs is a product lab for AI products, developer tools, technical writing, and future agent systems.

This repository is the v1 website for Bhawesh Labs. The first phase focuses on rebranding the original template while preserving the existing Next.js, TypeScript, and Tailwind foundation.

## Focus Areas

- AI products
- Developer tools
- AI agents
- Engineering writeups
- Open-source projects

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Run lint checks:

```bash
npm run lint
```
